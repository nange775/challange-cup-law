# 数据目录说明

**这是包含假数据的示例数据库，用于演示和测试。**

- **人员**：5个真实人物 + 2个空壳公司
  - 李建（嫌疑人）
  - 王强（嫌疑人）
  - 赵敏（涉案人）
  - 李辉（涉案人）
  - 张伟（证人）
  - 星辰商贸有限公司（企业）

  - 供述笔录：李建供述

  - 测谎报告：李建测谎结果
  - 起诉书：完整法律文书

### 如何使用

**方式1：直接使用（推荐）**
```bash
cd project
python server.py
# 访问 http://localhost:8001
# 数据已经在数据库中，可以直接查询分析
```

**方式2：重新生成数据**
```bash
cd project

# 清空数据库
rm -f data/investigation.db

# 重新生成假数据
python generate_fake_data.py

# 导入到数据库
python import_fake_data_to_db.py
```

### ⚠️ 重要说明

1. **这是虚构的假数据**，仅供系统演示和功能测试使用
2. 所有人名、案情、证据均为虚构，与真实案件无关
3. 真实使用时，请删除此数据库并导入真实数据
4. **不要在此数据库中存储真实敏感数据**

### 数据来源

基于 `data/fake data/李建徇私枉法案及证据.pdf` 中的虚构案例生成。

### 删除示例数据

如果要使用真实数据，删除此文件即可：
```bash
rm data/investigation.db
```

系统会在首次启动时自动创建空数据库。
